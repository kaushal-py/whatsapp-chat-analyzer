Whatsapp Chat Analysis in Python¶

Introduction¶

Whatsapp is a heavily used IM medium, especially by college students. Whatsapp Groups provide an easy medium to share data and news quickly. Many interesting insights might pop up, from a simple analyis on these chats.

Whatsapp includes a feature to email the history of a chat to a person. This feature can be exploited to do an analysis on the chats. This analysis notebook gives various insights that can be derived from the chat history of a Whatsapp group.

The following analysis was done on a college class group of 88 members, history dated from 28/08/16 to 03/06/17

Insights¶

Having a look at the file, we can see that the chats follow a format. We need to extract the data from that format somehow.

Also, we see that not all lines follow the format. Multiline messages are broken by new-line characters. This needs to be taken care of first. So we would be first bringing down the multiline messages to a single line, so that all the lines inthe file follow a standard format. This can be easily done with the help of Regex (Regular Expressions), which brings us to our next step.

Step 3: Data Wrangling (using regex)¶

Cleaning up the raw data is always a crucial part of any analysis as it decides what insights the data might lead you to. Removing out the unnecessary parts will bring about a significant improvement on the analysis.

Here, we use the python RE module (More about python RE module) to slice the different components from the text format.

Firstly, we define our Regular Expressions and compile them, so that they can be used later.

In [229]:
import re

# messages are matched using this re
message = re.compile(r'(\d{2}/\d{2}/\d{2,4}),\s(\d{1,2}:\d{2}\s[AaPp][Mm])\s-\s([^:]{1,30}):\s(.*)')

# notification messages such as changing the group name, icon are matched using this re
other = re.compile(
    r'(\d{2}/\d{2}/\d{2,4}),\s(\d{1,2}:\d{2}\s[AaPp][Mm])\s-\s.+')

Multiline to single line¶

The following piece of code, as the title suggests, converts the multiline messages to a single line and stores the data in a new Cleaned file.

In [230]:
file.seek(0)

fileWrite = open("D10ChatsCleaned.txt", "w", encoding="utf-8")

data = file.readlines()

for i, line in enumerate(data):
    m = message.match(line)
    n = other.match(line)
    if m or n:
        data[i] = "\n"+line.rstrip()
    else:
        data[i] = line.rstrip()

        
fileWrite.writelines(data)

fileWrite.close()

Text to Table¶

The cleaned text data can now be easily converted to tabular matrix format, using the same Regular Expressions that we used above. A (n X 4) matrix is created, which now contains 4 separate data columns; Date, Time, Sender and Message.

In [231]:
messages = []

fileCleaned = open("D10ChatsCleaned.txt", "r", encoding="utf-8")

for line in list(fileCleaned):
    m = message.match(line)
    if m:
        messages.append(list(m.groups()))

fileCleaned.close()

Step 5.3: When is the group active (Time of the day)¶

Another analysis we would like to do is during what time of the day are most of the messages sent. This can give various insights about the group.

Before going into the analysis, we convert the time fields, that are in string format to float values.

Float values can represent the time in 24 hour format. Eg. 0.12 signifies 12:12 AM and 12.45 denotes 12:45 PM

In [136]:
def strTimeToFloat(time):
    timeF = float(time[:-3].replace(":","."))
    if time[-2] == 'P' or time[-2] == 'p':
        if int(timeF) != 12:
            timeF+=12
    elif int(timeF) == 12:
        timeF -= 12
    return timeF

Here, we calculate the number of messages in each minute and sort it accorting to the time of the day (12 AM to 11.59 PM). We plot the data as a line plot between time and the number of messages sent.

Step 5.5: Characters per message¶

So far, we have done analysis on date and time. Now, we move on to the messages. We group the messages by the sender, and try to gain insight on the characters sent by each person.

We calculate the total number of characters sent by each person. This leads us to an interesting insight, what is the average length of message of each member. This can tell us if a person likes to keep his/her messages short or long.

For readability, only 10 members are displayed.

In [236]:
charPerMessage = []

for name, group in df["Message"].groupby(df["Sender"]):
    totLen = 0
    for message in group:
        totLen += len(message)
    charPerMessage.append([name, totLen, round(totLen/len(group), 2)])
